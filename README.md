# projet inf ba2
platformer 2D avec python/arcade

---

## Installation
1. cloner le repo avec:
```shell
git clone https://github.com/nono752/contre-projet.git
```
2. ouvrir terminal dans le clone
3. Lancer le jeu avec:
```shell
uv run main.py
```

## Autres commandes uv
* Lancer checker mypy
```shell
uv run mypy .
```

* Lancer les tests
```shell
uv run pytest
```
